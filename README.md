# Speech Practice App 🎤
 
An AI-powered public speaking practice web application. Record speeches, get real-time AI feedback on filler words, pacing, tone, and relevance, and track your progress over time
 
## What It Does
- User can record speeches on random topics
- Get AI feedback on filler words, pacing, tone, and relevance
- Track your progress over time


## Tech Stack
- Backend: FastAPI (Python)
- Frontend: React (JavaScript)
- Database: PostgreSQL
- AI: OpenAI API
